An env for Vue.

## Usage

```bash
# create a Vue workspace
bit new vue my-vue-workspace --aspect teambit.vue/vue # TODO: replace with your env id

# add a Vue component
bit create vue <component-name>

# add a Vue composable
bit create vue-composable <component-name>

# add a Vue app
bit create vue-app <component-name>

# add a custom Vue env
bit create vue-env <component-name>
```
